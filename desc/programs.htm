SWEgenskaper

Nyhet i version 0.44: Drag&drop från Utforskaren.

Varning! Att ändra filändelser mha programmet är livsfarligt!

Ett program som försöker att ändra attribut, datum, efternamn etc på ett gäng filer. Programmet är skrivet för experter. Nybörjare och barn kan utan tvekan fördärva en dator med några musklick.

Programmet kan köras i två moder...

...som ett Windowsprogram:



...och som ett DOS-program:



Skriver du exempelvis:

SWEgenskaper -t -z- -clm20021001010259 *.txt

vid DOS-promptern, så får alla txt-filer datumet (clm betyder "alla 3"): 2002-10-01 01:02:59. Vidare (-z-) så ändras alla txt-filer till gemena bokstäver. -t betyder att du vill göra detta även i alla "dottermappar".

BabbelFisken

Ett program som försöker att översätta diverse källkod till C#. Tyvärr fordras det ett "visst" manuellt efterarbete...

Följande källkod översätts:
  • Delphi
  • Turbo Pascal 5
  • VB.NET
  • Klassisk VB
...och snart FORTRAN.
